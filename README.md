I recently learned to use the UI package Avalonia and adapted my garage project to use it. 

Can create a simulated garage and add and remove vehicles to it. 
There are 5 categories of vehicles (Car, Motorcycle, Boat, Aiplane and Bus) which each have a unique value.
Uses drop down boxes and other UI features.

Boats and Airplanes take up 2 and 3 spaces in the garage and can therefore not be added if there isn't enough empty space in a row.
